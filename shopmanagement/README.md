# Shop Management Microservice
Manages supermarket outlet details for the B2B Supermarket Sales Management System.

## Prerequisites
- Docker and Docker Compose (version 2.0 or higher)
- Java 17
- Maven (version 3.8 or higher)
- PostgreSQL 15
